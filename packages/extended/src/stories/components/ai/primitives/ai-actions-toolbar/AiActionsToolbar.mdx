The AI Actions Toolbar provides a set of action buttons commonly used in AI chat interfaces for interacting with AI-generated content. It includes refresh, copy, and feedback actions that emit structured events for easy integration.

The component provides consistent styling using Forge design tokens and icon button patterns, with all functionality handled through event emission.

<Canvas of={AiActionsToolbarStories.Demo} />

> **Note:** All actions emit events for the parent application to handle. The component does not perform any functionality directly, allowing for flexible integration.

### Action Types

The toolbar provides four main actions:

- **Refresh** - Regenerate or refresh AI content
- **Copy** - Emit copy action event for parent to handle clipboard operations
- **Thumbs Up/Down** - Provide feedback on your AI responses

### Event Handling

The component emits a single event type for all actions:

```typescript
toolbar.addEventListener('forge-ai-actions-toolbar-action', event => {
  const { action } = event.detail; // 'refresh' | 'copy' | 'thumbs-up' | 'thumbs-down'

  switch (action) {
    case 'refresh':
      // Handle refresh request
      break;
    case 'copy':
      // Handle copy action (implement clipboard logic here)
      break;
    case 'thumbs-up':
    case 'thumbs-down':
      // Handle feedback
      break;
  }
});
```
