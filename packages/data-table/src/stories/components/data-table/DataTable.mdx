The Data Table component provides a powerful and flexible way to display and interact with tabular data.
It supports features such as sorting, filtering, column resizing and reordering, row selection, and more.

## Features

- **Sorting**: Click on column headers to sort data
- **Filtering**: Apply filters to columns to narrow down displayed data
- **Resizing**: Adjust column widths by dragging column dividers
- **Reordering**: Drag and drop columns to rearrange them
- **Row Selection**: Select single or multiple rows
- **Row Clicking**: Respond to row click events
- **Column Visibility**: Right-click headers to open a column menu or control visibility programmatically
- **Styling Options**: Apply various styling options like striped rows, hover effects, and more
- **Virtualization**: Efficiently handle large datasets with row virtualization for improved performance
- **Fixed Headers**: Keep column headers visible while scrolling through large datasets
