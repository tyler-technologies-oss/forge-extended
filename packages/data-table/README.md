# Tyler Forge™ Data Table

An advanced data table component for the Tyler Forge™ design system.

## Local Development

### Install

```bash
npm install
```

### Dev Site

You can also run a dev site using Vite:

```bash
npm run dev
```

### Build

This runs a production build:

```bash
npm run build
```

### Run tests

Execute the unit test suite:

```bash
npm test
```
